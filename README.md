# База данных ItemStorageBrowser

В этой ветке хранится и обновляется актуальный файл базы данных предметов на складах **ItemStorageDB.lua** (другие файлы в этой ветке пользователям не нужны, и остаются не актуальными).

## Инструкция по ручному обновлению БД складского аддона ItemStorageBrowser:

1. Скачивание файла **ItemStorageDB.lua** из этой ветки:
* нажмите на файл ItemStorageDB.lua выше;
* на открывшейся странице, справа вверху - нажмите кнопку со значком скачивания с подсказкой: "Download raw file";
* сохраните или перенесите скачанный в другое место файл **ItemStorageDB.lua** в папку игры с установленным аддоном:
   `\Interface\AddOns\ItemStorageBrowser\`, подтвердив замену старого файла.
2. Запустите игру, или перезайдите своим персонажем, или выполните команду в чате игры: `/reload`.

Теперь данные о складах будут обновлены в Вашем аддоне.

> **Примечания:**
> 1. Необходимо обновлять базу данных каждый раз перед тем, как использовать складской аддон: БД аддона в игре считается актуальной несколько минут при активности помощников по складу, или 1-2 часа при низкой их интенсивности работы со складом.
> 2. Для автоматического обновления БД складского аддона настоятельно **рекомендуется** использовать наше приложение [ItemStorage Manager](https://github.com/Phoenix-Guilds/ItemStorageDesktop). Это приложение работает в двух режимах:
> * **Админский режим** (включается только у помощников по складу, автоматически) позволяет обновлять БД складского аддона до нескольких раз в минуту;
> * **Пользовательский режим** (включается автоматически у всех остальных игроков) обновляет БД складского аддона раз в 15 минут.
